# Privacy Policy

## Filtro de Questões - SAVA

Last updated: May 31, 2026

Filtro de Questões - SAVA is a browser extension designed to help teachers visually filter question cards on the Sala de Avaliações platform.

This extension operates locally in the user's browser and does not collect, store, transmit, sell, or share personal data.

---

## Data Collection

This extension does **not** collect any user data.

Specifically, it does not collect:

* personal information;
* login credentials;
* browsing history;
* student data;
* teacher data;
* assessment data;
* question content;
* selected question IDs;
* cookies;
* authentication tokens;
* analytics data;
* usage statistics.

---

## Data Storage

This extension does **not** store user data.

It does not use:

* external databases;
* remote servers;
* analytics services;
* cloud storage;
* localStorage;
* sessionStorage;
* cookies.

All filtering actions occur temporarily in the current browser tab and are lost when the page is reloaded.

---

## Data Transmission

This extension does **not** send data to any external server.

It does not make external network requests and does not communicate with third-party APIs.

All processing is performed locally in the browser.

---

## Permissions

The extension uses limited permissions only to provide its intended functionality.

### `activeTab`

Used to allow the extension to interact with the active tab when the user clicks the extension icon.

### `scripting`

Used to inject or reload the extension interface in the active page when necessary.

### Host permission

The extension is restricted to the following domain:

```text
https://admin.saladeavaliacoes.com.br/*
```

The extension does not use broad access permissions such as `<all_urls>`.

---

## How the Extension Works

The extension adds a visual panel to the question selection page.

Through this panel, the user can:

* filter visible question cards by origin;
* filter visible question cards by text;
* search through paginated question lists;
* temporarily hide question cards that do not match the selected filter;
* highlight matching question cards;
* optionally click visible question selection buttons.

These operations only modify the visual presentation of the page in the user's browser.

The extension does not alter the platform database and does not directly submit, edit, or delete assessment data.
